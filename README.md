📊 Financial Compliance Risk Analysis

Interactive Excel dashboard analyzing the relationship between Total Audit and Fraud using statistical analysis and visual reporting. The project combines regression analysis, histogram distribution, correlation and covariance analysis, and descriptive statistics to highlight financial risk and compliance insights.

📊 Key Features
KPI-style dashboard summarizing key statistical outputs
Regression Analysis to study the relationship between Total Audit and Fraud
Histogram to visualize data distribution patterns
Correlation & Covariance analysis for relationship strength and direction
Descriptive Statistics for central tendency, spread, and variability
Clean and aesthetic Excel dashboard design for portfolio, resume, and GitHub showcase

📂 Dataset
Dataset sourced from Kaggle: Big 4 Financial Risk Compliance

🛠 Tools
Excel, Data Analysis ToolPak
